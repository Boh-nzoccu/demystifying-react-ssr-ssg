# Demystyfing React SSR/SSG

## What is SSR?

SSR stands for Server Side Rendering

## What is SSG?

SSG stands for Static Site Generation

## Try SSG with this project

1. Clone this repo
2. Run `pnpm install`
3. Run `pnpm ssg && pnpm serve` 


## Try SSR with this project

1. Clone this repo
2. Run `pnpm install`
3. Run `pnpm start`
